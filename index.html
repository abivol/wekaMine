wekaMine

The missing piece for Weka.

About

wekaMine fills many of the usability holes in the Weka machine learning library:
  • Much easier to run large experiments with wekaMine.
  • Standardized basic pipeline.
  • Compact specification of search over algorithms and parameters.
  • Additional algorithms.
  • Feature score outputs from CV.
  • Lots of sugar for working with Weka from code (e.g. instance ID support)
  • Serialized models encapsulating whole pipeline.
